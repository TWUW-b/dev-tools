# 管理者ログイン
- 管理者アカウントでログインできる
- 一般ユーザーとは別画面に遷移する
- 権限のないユーザーはアクセス拒否される

# ユーザー管理
- ユーザー一覧が表示される
- ユーザーを検索・絞り込みできる
- ユーザーを新規作成できる
- ユーザー情報を編集できる
- ユーザーを無効化できる
- パスワードリセットを代行できる

# コンテンツ管理
- コンテンツ一覧が表示される
- コンテンツを新規作成できる
- コンテンツを編集・削除できる
- 公開/非公開を切り替えられる

# 承認フロー
- 承認待ち一覧が表示される
- 内容を確認して承認できる
- 差戻しでコメントを残せる

# レポート
- 期間指定でレポートを出力できる
- CSV でエクスポートできる
- グラフで可視化される
