# 01 管理者ログイン
- [TC-AD-001] 管理者アカウントでログインすると管理画面トップが表示される
- [TC-AD-002] ログイン後のヘッダに管理者モードであることを示すバッジが表示される

# 02 ユーザー管理
- [TC-AD-101] ユーザー一覧画面にユーザーが表示される
- [TC-AD-102] 検索ボックスにメールアドレスを入力すると一覧が絞り込まれる
- [TC-AD-103] 新規作成ボタンからユーザーを追加すると一覧に新しい行が出現する
- [TC-AD-104] ユーザー編集画面で名前を変更して保存すると一覧の表示名が更新される
- [TC-AD-105] 無効化ボタンを押すと該当ユーザー行に「無効」バッジが表示される
- [TC-AD-106] パスワードリセット代行ボタンを押すと完了モーダルが表示される

# 03 コンテンツ管理
- [TC-AD-201] コンテンツ一覧画面にコンテンツが表示される
- [TC-AD-202] 新規作成フォームから保存するとコンテンツ一覧に新しい行が出現する
- [TC-AD-203] コンテンツ編集画面で保存すると一覧の表示が更新される
- [TC-AD-204] 削除ボタンで確認ダイアログを経由して削除すると一覧から行が消える
- [TC-AD-205] 公開トグルを切り替えると一覧の公開状態バッジが更新される

# 04 承認フロー
- [TC-AD-301] 承認待ち一覧画面に保留中項目が表示される
- [TC-AD-302] 承認ボタンを押すと行のステータス表示が「承認済み」に変わる
- [TC-AD-303] 差戻しモーダルでコメントを入力して送信すると行が「差戻し」になる

# 05 レポート
- [TC-AD-401] レポート画面で期間を指定するとグラフが画面上に描画される
- [TC-AD-402] CSV エクスポートボタンを押すとブラウザのダウンロードが開始する
