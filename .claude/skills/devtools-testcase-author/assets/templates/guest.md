# 公開ページ閲覧
- トップページが表示される
- プライバシーポリシー・利用規約が閲覧できる
- お問い合わせフォームが表示される

# サインアップ
- 新規アカウントを作成できる
- 既存メールアドレスで登録しようとするとエラーになる
- パスワード強度が不足するとエラーになる
- 確認メールが届く

# ログイン
- 登録済みメールとパスワードでログインできる
- 存在しないメールでログイン試行するとエラーになる
- パスワード誤入力でエラーになる
- パスワードリセットを開始できる

# アクセス制限
- 保護ページにアクセスするとログインへリダイレクトされる
- URL 直打ちでもリダイレクトが効く
