# 01 ログイン
- [TC-CL-001] 発行されたクライアント ID とパスワードでログインすると自社ダッシュボードが表示される
- [TC-CL-002] 初回ログイン時にパスワード変更画面が自動で表示される
- [TC-CL-003] パスワード変更完了後にダッシュボードへ遷移する

# 02 自社データ閲覧
- [TC-CL-101] 一覧画面を開くと自社に紐づく案件のみが表示される
- [TC-CL-102] 一覧の各行をクリックすると詳細画面に遷移する
- [TC-CL-103] 詳細画面でステータス・担当者・更新日時が画面に表示される

# 03 承認・差戻し
- [TC-CL-201] 承認待ちリスト画面に承認待ち案件が表示される
- [TC-CL-202] 承認ボタンを押すと該当行のステータス表示が「承認済み」に変わる
- [TC-CL-203] 差戻しモーダルでコメントを入力して送信するとモーダルが閉じて行が「差戻し」表示になる

# 04 帳票ダウンロード
- [TC-CL-301] PDF ダウンロードボタンを押すとブラウザの標準ダウンロードが開始する
- [TC-CL-302] Excel ダウンロードボタンを押すと同じくダウンロードが開始する
- [TC-CL-303] ダウンロード履歴画面に直近のダウンロード一覧が表示される

# 05 フィードバック
- [TC-CL-401] 問い合わせフォームに内容を入力して送信すると完了メッセージが表示される
- [TC-CL-402] 問い合わせ履歴画面に自分が送信した問い合わせの一覧が表示される
