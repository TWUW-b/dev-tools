# ログイン
- 発行されたクライアント ID でログインできる
- 初回ログイン時にパスワード変更を求められる

# 自社データ閲覧
- 自社に紐づくデータのみ一覧表示される
- 他社データが一切見えない
- 詳細画面に遷移できる

# 承認・差戻し
- 承認待ちリストが表示される
- 承認ボタンでステータスが変わる
- 差戻しコメントを残せる

# 帳票ダウンロード
- PDF をダウンロードできる
- Excel をダウンロードできる
- ダウンロード履歴が残る

# フィードバック
- 問い合わせを送信できる
- 送信した問い合わせの履歴が見える
