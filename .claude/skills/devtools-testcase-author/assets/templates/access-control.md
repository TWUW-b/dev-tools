# 01 企業・テナント境界
- [TC-AC-001] 企業管理者: 他企業の管理画面 URL を直打ちしても該当画面が開かず、エラーかログイン画面にリダイレクトされる
- [TC-AC-002] 企業管理者: 他企業のユーザー編集 URL を直打ちしても編集フォームが表示されない
- [TC-AC-003] 一般ユーザー: 共有されていない他企業のファイル詳細 URL に直接アクセスしても画面が開かない
- [TC-AC-004] 一般ユーザー: 他企業の案件一覧 URL を直打ちしても自企業の案件一覧に戻される

# 02 ゲスト認可
- [TC-AC-101] ゲスト: 保護された画面（/dashboard 等）に直接アクセスするとログイン画面に遷移する
- [TC-AC-102] ゲスト: 公開リンク URL を踏んだ後、ログインが必要なアクションを試みるとログインモーダルが出る
- [TC-AC-103] ゲスト: 管理者専用画面（/admin 等）に直接アクセスするとログイン画面に遷移する

# 03 権限境界（読み取り専用）
- [TC-AC-201] 閲覧ユーザー: /corporate の編集ボタンが画面上に表示されない
- [TC-AC-202] 閲覧ユーザー: /user/add のような招待画面にアクセスしても「権限がありません」と表示される
- [TC-AC-203] 閲覧ユーザー: ファイル削除ボタンが表示されず、URL 直打ちでも削除画面が開かない

# 04 権限昇格の拒否
- [TC-AC-301] 一般ユーザー: 自分のロールを管理者に変更する UI が存在しない
- [TC-AC-302] 一般ユーザー: 他人のロール編集ボタンが表示されない
- [TC-AC-303] 一般ユーザー: 管理者専用メニューがサイドバーに表示されない

# 05 セッション・認証
- [TC-AC-401] ログアウト後: 戻るボタンでブラウザ履歴を戻しても、保護画面は再表示されずログイン画面になる
- [TC-AC-402] トークン期限切れ: 画面操作時に自動的にログイン画面へ遷移する
- [TC-AC-403] 別ユーザーでのログイン: 前のユーザーのデータが画面に残らず、自分のデータのみが表示される
