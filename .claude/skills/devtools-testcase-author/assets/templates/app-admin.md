# 環境設定
- アプリ全体の設定画面にアクセスできる
- メール送信設定を変更できる
- 外部 API キーを登録・更新できる
- 設定変更が即時反映される

# 権限・ロール管理
- ロール一覧が表示される
- 新しいロールを作成できる
- ロールに権限を割り当てられる
- ユーザーにロールを付与・剥奪できる

# 組織・テナント管理
- テナント一覧が表示される
- テナントを新規作成できる
- テナントを停止・再開できる

# システムログ
- 操作ログを閲覧できる
- 期間・ユーザーで絞り込める
- エラーログを閲覧できる

# バックアップ・メンテナンス
- 手動バックアップを実行できる
- バックアップ履歴が確認できる
- メンテナンスモードの ON/OFF ができる
- メンテナンス中は一般ユーザーがアクセス不可になる

# 監査
- 重要操作の監査ログが残る
- 監査ログはエクスポート可能
