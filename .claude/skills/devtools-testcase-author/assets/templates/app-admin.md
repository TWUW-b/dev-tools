# 01 環境設定
- [TC-AP-001] アプリ設定画面が開ける
- [TC-AP-002] メール送信設定画面で値を変更して保存すると同画面上で新しい値が表示される
- [TC-AP-003] 外部 API キー登録フォームで値を入力して保存すると一覧に新しい行が出現する
- [TC-AP-004] 設定変更後に対象機能を操作すると新しい設定が反映された画面が表示される

# 02 権限・ロール管理
- [TC-AP-101] ロール一覧画面にロールが表示される
- [TC-AP-102] 新規ロール作成フォームから保存するとロール一覧に新しい行が出現する
- [TC-AP-103] ロール編集画面で権限チェックボックスを変更して保存するとロール詳細の権限表示が更新される
- [TC-AP-104] ユーザー詳細画面でロール割当を変更して保存するとユーザー詳細のロール表示が更新される

# 03 組織・テナント管理
- [TC-AP-201] テナント一覧画面にテナントが表示される
- [TC-AP-202] 新規テナント作成フォームから保存するとテナント一覧に新しい行が出現する
- [TC-AP-203] テナント詳細画面で停止ボタンを押すと一覧の該当行に「停止」バッジが表示される
- [TC-AP-204] 再開ボタンを押すと「停止」バッジが消える

# 04 システムログ
- [TC-AP-301] 操作ログ画面にログ行が表示される
- [TC-AP-302] 期間フィルタを変更すると一覧が絞り込まれる
- [TC-AP-303] ユーザーフィルタでメールアドレスを指定すると一覧が該当ユーザーに絞り込まれる
- [TC-AP-304] エラーログタブを開くとエラーメッセージ一覧が表示される

# 05 バックアップ・メンテナンス
- [TC-AP-401] 手動バックアップボタンを押すと完了モーダルが表示される
- [TC-AP-402] バックアップ履歴画面に直近のバックアップ一覧が表示される
- [TC-AP-403] メンテナンスモード ON トグルを押すとヘッダに「メンテナンス中」バナーが表示される
- [TC-AP-404] メンテナンスモード中に別ブラウザから一般ユーザーでアクセスすると「メンテナンス中」画面が表示される

# 06 監査
- [TC-AP-501] 監査ログ画面に重要操作の一覧が表示される
- [TC-AP-502] 監査ログのエクスポートボタンを押すとブラウザのダウンロードが開始する
