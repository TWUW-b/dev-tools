# ログイン・アカウント
- メールアドレスとパスワードでログインできる
- パスワードリセットメールを受け取れる
- ログアウトできる
- ログイン状態が保持される（再訪時）

# ダッシュボード
- ログイン直後にダッシュボードが表示される
- 未読通知の件数が表示される
- 最近のアクティビティが表示される

# プロフィール
- 自分のプロフィールを閲覧できる
- プロフィール情報を編集できる
- アバター画像をアップロードできる
- パスワードを変更できる

# データ登録
- 新規データを登録できる
- 必須項目未入力でエラーメッセージが表示される
- 登録後に一覧へ反映される

# データ編集・削除
- 自分が作成したデータのみ編集できる
- 他人のデータは編集ボタンが出ない
- 削除時に確認ダイアログが出る
- 削除後に一覧から消える

# 通知
- 通知一覧が閲覧できる
- 通知を既読にできる
- 通知設定を変更できる
