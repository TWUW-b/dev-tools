# 01 ログイン・アカウント
- [TC-US-001] ログイン画面からメールアドレスとパスワードでログインするとダッシュボードが表示される
- [TC-US-002] パスワードリセット画面からメールアドレスを送信すると送信完了メッセージが表示される
- [TC-US-003] ヘッダメニューからログアウトするとログイン画面に戻る
- [TC-US-004] 一度ログインした後にブラウザタブを閉じて再度開くと再ログインなしでダッシュボードが表示される

# 02 ダッシュボード
- [TC-US-101] ログイン直後にダッシュボードが表示される
- [TC-US-102] ヘッダまたはサイドバーに未読通知の件数バッジが表示される
- [TC-US-103] ダッシュボードに最近のアクティビティ一覧が表示される

# 03 プロフィール
- [TC-US-201] プロフィール画面に自分の名前とメールアドレスが表示される
- [TC-US-202] プロフィール編集画面で名前を変更して保存すると、その場で表示名が更新される
- [TC-US-203] アバター画像をアップロードすると画像が即時プレビューに反映される
- [TC-US-204] パスワード変更画面で変更後、完了メッセージが表示される

# 04 データ登録
- [TC-US-301] 新規登録画面から必須項目を入力して保存すると一覧画面に新しい行が出現する
- [TC-US-302] 必須項目未入力で保存ボタンを押すとエラーメッセージが入力欄下に表示される
- [TC-US-303] 登録完了後にトースト通知が表示される

# 05 データ編集・削除
- [TC-US-401] 自分が作成したデータ行に編集ボタンが表示される
- [TC-US-402] 他人が作成したデータ行には編集ボタンが表示されない
- [TC-US-403] 削除ボタンを押すと確認ダイアログが表示される
- [TC-US-404] 確認ダイアログで削除を選ぶと対象行が一覧から消える

# 06 通知
- [TC-US-501] 通知一覧画面が開ける
- [TC-US-502] 通知をクリックすると既読マークが付き、未読件数バッジが減る
- [TC-US-503] 通知設定画面で ON/OFF を切り替えて保存すると設定画面上で状態が維持される
